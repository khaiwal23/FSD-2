Learning Outcomes
Gained a clear understanding of backend web development using the Flask framework in Python, including application structure and routing mechanisms.
Learned how to establish a connection between a Flask application and a MySQL database, enabling seamless interaction between the application layer and the data layer.
Acquired practical knowledge of designing and creating relational database tables in MySQL, including defining attributes, primary keys, and constraints such as NOT NULL and UNIQUE.
Developed the ability to implement complete CRUD (Create, Read, Update, Delete) operations using RESTful API endpoints, allowing efficient data management.
Understood how to handle HTTP methods such as GET, POST, PUT, and DELETE and their role in performing different operations in a web application.
Implemented input validation techniques to ensure data accuracy and integrity, such as checking for empty fields, validating numerical inputs, and enforcing unique constraints.
Gained hands-on experience in sending and receiving JSON data between client and server, and learned how Flask processes request data using request.get_json() and returns responses using jsonify().
Learned how to test and debug REST APIs effectively using tools like Postman, including sending requests, analyzing responses, and identifying errors.
Improved understanding of error handling and debugging techniques in backend applications, including resolving issues related to database connectivity and invalid requests.
Developed skills in integrating multiple technologies (Flask, MySQL, and API testing tools) to build a complete working application.
Gained practical exposure to real-world backend development workflows, including project setup, coding, testing, and documentation.
Learned how to document the project using a README file, including describing features, setup instructions, and API endpoints.
Enhanced problem-solving and logical thinking skills through debugging errors such as connection issues, incorrect routes, and invalid inputs.
Understood the importance of API design and structure in building scalable and maintainable web applications.
